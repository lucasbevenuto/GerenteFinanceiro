# 💰 Gerente Financeiro API

Sistema de gerenciamento financeiro pessoal desenvolvido com foco em boas práticas de arquitetura backend utilizando .NET.

A aplicação permite o controle de receitas e despesas, categorização de transações e geração de relatórios financeiros, além de integração com APIs externas para consulta de cotação de moedas em tempo real.

## 🚀 Funcionalidades

- Cadastro de usuários
- Registro de receitas e despesas
- Categorização de transações
- Relatórios financeiros mensais
- Integração com API externa de cotação de moedas
- Estrutura baseada em Clean Architecture

## 🛠️ Tecnologias utilizadas

- C#
- .NET (ASP.NET Core Web API)
- Entity Framework Core
- SQL Server
- REST APIs
- Git

## 🧱 Arquitetura

O projeto foi estruturado utilizando princípios de Clean Architecture, separando responsabilidades em camadas:

- **Domain** → Entidades e regras de negócio
- **Application** → Casos de uso e serviços
- **Infrastructure** → Acesso a dados e integrações externas
- **API** → Exposição dos endpoints

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento backend com .NET, modelagem de dados, consumo de APIs externas e organização de código em arquitetura escalável.
